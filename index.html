Acquire Token

Click the button below to acquire a token. The token will expire after a set period.

Warning: If you lose your token, it cannot be recovered. You will need to create a new one.

API Documentation

Create/Update Achievement

POST https://your-api-endpoint.amazonaws.com/achievement
{
    "action": "create",
    "userId": "string",
    "token": "string",
    "name": "string",
    "description": "string",
    "goal": "number",
    "details": "string",
    "progress": "number",
    "dateAchieved": "string"
}

Get Achievements List

GET https://your-api-endpoint.amazonaws.com/achievement
{
    "action": "get_list",
    "token": "string"
}

Get User Achievement

GET https://your-api-endpoint.amazonaws.com/achievement
{
    "action": "get_user_achievement",
    "userId": "string",
    "token": "string",
    "name": "string"
}

Get Achievement

GET https://your-api-endpoint.amazonaws.com/achievement
{
    "action": "get_achievement",
    "token": "string",
    "name": "string"
}

Delete Achievement

DELETE https://your-api-endpoint.amazonaws.com/achievement
{
    "action": "delete",
    "userId": "string",
    "token": "string",
    "name": "string"
}
